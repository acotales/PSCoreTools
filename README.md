# PSCoreTools

Reusable core utility functions for PowerShell automation projects.

---

## Installation

Clone the repository:

```powershell
git clone https://github.com/acotales/PSCoreTools.git
```

Import the module:

```powershell
Import-Module .\PSCoreTools\PSCoreTools.psd1
```

---

## Usage

Import the module:

```powershell
Import-Module PSCoreTools
```

List available commands:

```powershell
Get-Command -Module PSCoreTools
```
